# ToolsQA Selenium Automation Project

## Overview
This project automates functional and end-to-end test cases for the ToolsQA Demo website using Selenium WebDriver, TestNG, and Maven.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Chrome & Firefox
- GitHub Actions (CI)

 ## Test Coverage
- User Registration
- Login (Positive & Negative)
- Navigation
- Delete Account
- Cross-browser testing

## Project Structure
- src/test/java → Test cases, Page Objects, Test data
- testng.xml → Test suite configuration
- pom.xml → Maven dependencies

## How to Run Tests

### Run locally
```bash
mvn clean test
